# Vivid-Gallery-App-
Vivid Gallery is a modern image gallery web application featuring Local Gallery, Global Gallery (Pexels API), image uploads, post creation, and AI-powered image search with a responsive, user-friendly interface.
# 🌄 Vivid Gallery
Vivid Gallery is a modern Progressive Web Application (PWA) that offers an interactive image browsing and sharing experience. It combines a personal Local Gallery with a Global Gallery powered by the Pexels API, allowing users to upload, explore, and search images through a clean and responsive interface.

## ✨ Features

- 🖼️ Local Gallery for managing personal images
- 🌍 Global Gallery with high-quality images from the Pexels API
- 📤 Upload images from your device
- 📝 Create and manage image posts
- 🔍 Image search using image scanning
- 📱 Fully responsive design for desktop, tablet, and mobile
- ⚡ Progressive Web App (PWA) support
- 🌙 Modern, clean, and user-friendly interface
- 🚀 Fast performance with Service Worker caching

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Pexels API
- Progressive Web App (PWA)
- Service Worker
