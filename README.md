🎨 Vivid Gallery

A modern, responsive web-based gallery application built using HTML, CSS, and JavaScript. Vivid Gallery provides a clean, elegant, and user-friendly interface for exploring beautiful image collections across desktop and mobile devices.

🌐 Live Demo

Website: https://vivid-gallery-app-ayxq.vercel.app/

✨ Features

- 📱 Fully responsive design
- 🖼️ Beautiful and interactive image gallery
- ⚡ Fast loading and smooth performance
- 🎨 Modern and clean user interface
- 📲 Progressive Web App (PWA) support
- 🌍 Cross-browser compatibility

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Progressive Web App (PWA)
- Git & GitHub
- Vercel

🚀 Getting Started

Clone the repository

git clone https://github.com/Subhi000/vivid-gallery.git

Navigate to the project folder

cd vivid-gallery

Run the project

Simply open index.html in your preferred web browser, or serve it using any local development server.
